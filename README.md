## Antask
ブラウザ上でタスク管理ができるシンプルなアプリケーションです。

# Demo
![f2b8fe5a5b4d10c7431143078c0ebb3f](https://user-images.githubusercontent.com/52348865/70857502-07a0e700-1f33-11ea-96a1-39138df5b5a8.png)

# Characteristics
* タスク登録・編集時に確認画面を表示
* Bootstrap/slimを初使用
* テストコードはRSpecを利用

# Environment
* ruby 2.5.1
* rails 5.2.3
